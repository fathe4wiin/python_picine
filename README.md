# python_picine
## Module 00 - Python Fundamentals

### Topics Covered

#### 1. **Basic Output**
- `print()` function for displaying text
- String messages and formatted output

#### 2. **Input & Data Types**
- `input()` function for user input
- Type conversion: `int()`, `str()`, `float()`
- Working with integers and strings

#### 3. **Variables & Arithmetic Operations**
- Variable assignment and naming
- Basic arithmetic: addition (+), multiplication (*)
- Mathematical expressions in code

#### 4. **String Formatting**
- f-strings for dynamic string output
- String interpolation with variables

#### 5. **Control Flow - Conditionals**
- `if` statements for decision-making
- `if-else` blocks for branching logic
- Comparison operators (>, <, ==)
- Boolean conditions

#### 6. **Control Flow - Loops**
- `while` loops for repeated execution
- Loop counters and increments (+=)
- Loop control with conditional breaks

#### 7. **Functions**
- Function definition with `def`
- Function parameters and arguments
- Return types with `-> None` type hints
- Function calls and execution

#### 8. **Type Hints**
- Parameter type annotations (e.g., `seed_type: str`)
- Return type annotations (e.g., `-> None`)
- Best practices for code clarity

### Key Concepts Learned
- Procedural programming basics
- User interaction and input handling
- Logical decision-making
- Repetition and automation
- Code organization through functions
- Python conventions and type safety
